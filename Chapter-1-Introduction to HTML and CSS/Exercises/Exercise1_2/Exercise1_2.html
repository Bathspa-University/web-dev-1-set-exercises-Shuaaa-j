Paragraph

This is an HTML & CSS Quick Reference Guide for Web Development.

Code: <p>This is an HTML & CSS Quick Reference Guide for Web Development.</p>

Usage: Use the <p> tag to create paragraphs of text. Paragraphs are commonly used to present textual content on a web page.

Ordered List

  1. Item 1
  2. Item 2
  3. Item 3
Code: <ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

Usage: Use the <ol> tag to create an ordered list. Ordered lists are used when you want to present a list of items in a specific sequence or order.

Unordered List

  • Item 1
  • Item 2
  • Item 3
Code: <ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Usage: Use the <ul> tag to create an unordered list. Unordered lists are used when you want to present a list of items without any specific sequence or order.
